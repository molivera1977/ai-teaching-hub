Step 1 of 5 — Know Your File Types

🚀 Before you begin: Open one of these in a new tab — you'll use it to practice saving and printing by the end of this lesson.
Claude or ChatGPT or Gemini or Copilot

📊 Which AI Tool Creates Which File Type?

Once AI creates something great for you, the next step is getting it into a format you can actually use — printed, shared, or saved as a file. But not every AI tool handles every file type equally — and you have four main options to choose from. Here's a quick breakdown of each:

Quick guide to the four file types:

📄 Google Doc
Opens in Google Drive in your browser. Easy to edit, easy to share with a link. Great if your school uses Google tools. Free.
📝 Word Doc (.docx)
Opens in Microsoft Word. Fully editable — change wording, fonts, layout. Standard for most schools that use Microsoft. Requires the Word app or Office 365.
📕 PDF
Opens on any device exactly the same way. Locked layout — students can't accidentally change it. Best for handouts, rubrics, and anything you want to look polished and final.
🌐 HTML (web page)
A web page file. Double-click to open in any browser (Chrome, Safari, Edge), then print. Works on every device with no app needed. The universal fallback when other formats fail.

What this chart means for you:

  • Use the tool that matches the file you need. If you need a Google Doc, use Gemini. If you live in Microsoft Word, Copilot is your most natural fit. ChatGPT is the most reliable all-around choice for Word docs and PDFs.
  • No AI tool makes Google Docs except Gemini. Everyone else can only give you Word, PDF, or HTML.
  • HTML works in every tool — it's the universal fallback. If a tool isn't great at Word or PDF, just ask for HTML and print from your browser.
  • You don't need to memorize this. Just keep this lesson bookmarked — when you need a file type, glance at the chart to pick the right tool.
✋ Good news — this is easier than you think. You don't have to figure out how to save anything yourself. Just ask AI to make the file for you — type something like "Now make this into a Word doc" and AI hands you a download link. That's the method most teachers will use most of the time, and we'll walk through exactly how on the next page.

Step 2 of 5 — Method 1: Ask AI for the File

⭐ Method 1 — Just Ask AI to Create the File

This is the easiest approach, and it's probably what you'll use most. After AI gives you a response, you can ask it to put everything into a Word document, Google Doc, or PDF — fully formatted and ready to print.

  1. Use AI to create your content first — a quiz, rubric, lesson plan, or anything else.
  2. Once AI gives you the response, left-click Copy on one of the prompts below and paste it into the same chat.
  3. Replace any [brackets] with your specifics, then hit Enter.
  4. AI will generate the file and show you a download link or button in the chat. Left-click it.
  5. The file saves to your Downloads folder. Open it, review it, and print or share.
✅ Best overall option — ChatGPT (free and paid plans can do Word docs, PDFs, and HTML files)

Ask ChatGPT for a Word doc, PDF, or HTML file

ChatGPT can generate a real file and give you a download link to left-click. No copy-paste, no extra steps — it just hands you the finished file.

Ask for a Word doc
Please create a downloadable Word document of this quiz. Format it with a title at the top, a Name and Date line, numbered questions with space to write answers, and an answer key on the last page.
Ask for a PDF
Please create a downloadable PDF of this rubric. Format it as a clean table with the assignment title at the top and a space for the student's name.
Ask for an HTML file
Please create this as a printable HTML file — clean layout, title at the top, Name and Date line, numbered questions with writing space, and an answer key at the end.

ChatGPT will show a download link in the chat — it looks like a blue underlined filename (e.g., quiz.docx or rubric.pdf) or a button that says "Download." Left-click it and the file goes straight to your Downloads folder. Open and print.

✅ Google Gemini (only option that can do Google Docs)

Ask Gemini to create a Google Doc or HTML file

Gemini's superpower is Google Docs — it can push a document directly into your Google Drive with one left-click, already formatted and ready to print. If your school uses Google tools, this is the smoothest workflow you'll find.

Ask for a Google Doc
Please create this as a Google Doc I can open and print. Format it with a title, a Name and Date line, numbered questions with writing space, and an answer key at the bottom.
Ask for an HTML file
Please create this as a printable HTML file — clean layout, title at the top, Name and Date line, numbered questions with writing space, and an answer key at the end.

Gemini will create the Google Doc in your Drive — just open it and print. Note: Gemini is hit or miss with Word docs and PDFs, so stick to Google Docs or HTML for the most reliable results.

✅ Easy to download — Claude · Free plan (HTML files only) · Paid plan (Word docs and PDFs)

Ask Claude for a Word doc, PDF, or HTML file

Claude shows your file in a preview panel right in the chat — left-click the download button and it's yours. HTML files are available on the free plan and print beautifully from any browser. Word and PDF downloads require a Claude Pro subscription (~$20/month), but HTML gets the job done just as well for printing.

Ask for a Word doc
Please create this as a Word document — clean and print-ready, with a title at the top, a Name/Date line, numbered questions with writing space, and an answer key at the end.
Ask for a PDF
Please create this as a downloadable PDF — clean and print-ready, with a title at the top, a Name/Date line, numbered questions with writing space, and an answer key at the end.
Ask for an HTML file
Please create this as a printable HTML file — clean layout, title at the top, Name and Date line, numbered questions with writing space, and an answer key at the end.

Claude will give you the file right in the chat. Left-click the link or attachment to download it, then open and print.

✅ Microsoft Copilot — home to Word docs with seamless integration · Can also produce PDFs and HTML files

Ask Copilot for a Word doc or PDF

If your school uses Microsoft 365, Copilot is your best friend for Word documents — it can create them natively right inside Microsoft Word. You can also use it at copilot.microsoft.com in any browser.

Ask for a Word doc
Please create this as a Word document — clean and print-ready, with a title at the top, a Name/Date line, numbered questions with writing space, and an answer key at the end.
Ask for a PDF
Please create this as a downloadable PDF — clean and print-ready, with a title at the top, a Name/Date line, numbered questions with writing space, and an answer key at the end.

Copilot will give you a downloadable file right in the chat. If you have Microsoft 365 Copilot in Word, you can even generate it there and it opens directly in Word — ready to print or share via Teams.

🔍 What about Perplexity? Perplexity is a fantastic research and image generation tool (see Lesson 4!), but it works differently when it comes to file creation. If you ask Perplexity to create a Word doc or PDF, it will write the content for you — but it doesn't generate an actual downloadable file. You'd still have to copy the text into Word or Google Docs yourself. For saving and printing, stick with ChatGPT, Gemini, Claude, or Copilot.

Step 3 of 5 — Method 2: Print from HTML

🌐 Method 2 — Printing Directly from an HTML File

This is a method most teachers don't know about — and once you try it, you may never go back to Word docs for certain things.

When AI creates a worksheet, quiz, or handout as an HTML file (a web page file), you can open it in any browser and print it directly. No Word, no Google Docs, no formatting to fix.

Pros and cons of printing HTML files:

✅ Pros
  • Looks the same every time — no font shifts, no weird margins, no layout surprises.
  • No app needed — opens in any browser on any device (school computer, Chromebook, laptop).
  • Faster to share — send it to a colleague and it looks identical on their screen.
  • No accidental edits — students can't mess with the formatting if you hand it out digitally.
  • Beautifully designed — columns, boxes, color sections, and tables come out clean (painful to do manually in Word).
⚠️ Cons
  • Not editable like a Word doc — if you want to tweak wording later, you'll need to ask AI for a new version.
  • Extra step the first time — you have to open the file in a browser, then go to File → Print (instead of just hitting print from Word).
  • Backgrounds may not print by default — colored boxes and section backgrounds might come out white unless you enable "Background graphics" in the print dialog.
  • Looks like a webpage on screen — students sometimes assume it's a website link, not a printable document. A quick "this is a worksheet to print" helps.
  • Some schools block .html files in email attachments. Print first, or share through Google Classroom or Teams.

How to print an HTML file:

1Ask your AI to create the worksheet or handout "as an HTML file."

2Get the file. What you see next depends on which AI you used:

Claude: The page appears in a panel on the right (called an Artifact). Left-click the Copy button at the top → choose Download as HTML.
ChatGPT: A download link appears in the chat — looks like a blue filename (worksheet.html). Left-click to download.
Gemini, Copilot, or Perplexity: You'll see code in the chat instead of a download link. Left-click Copy code, open Notepad (Windows) or TextEdit (Mac, set to plain text), paste, and save the file as worksheet.html.
Tip: You can also ask any AI "Can you give me a download link for that?" — it works in ChatGPT and Claude, but Gemini, Copilot, and Perplexity may just give you the code again.

3Double-click the file — it opens automatically in your browser (Chrome, Safari, or Edge).

4Press Ctrl+P (Windows) or Cmd+P (Mac) to open the print dialog.

5Choose your printer and left-click Print. Done.

Ask AI for a printable HTML worksheet
Please create this quiz as a printable HTML file. Use a clean layout with the title at the top, a Name and Date line, numbered questions with space to write answers, and an answer key section at the bottom. Make it look professional and easy to read when printed.

Step 4 of 5 — Method 3 & Quick Guide

🖼️ Method 3 — Saving AI-Generated Images

When AI creates an image (from Lesson 4!), saving it is simple:

1Hover over the image in the chat — a download icon ⬇️ usually appears. Left-click it.

2If no icon appears, right-click the image — on a Mac, tap with two fingers on your trackpad, or hold the Control key and left-click — then choose "Save image as…"

3Save it to your Desktop so it's easy to find.

Once saved, you can add it to a Google Doc or Word document — open your document, left-click the word Insert in the menu bar at the top, choose Image, then choose Upload from computer and find the file you just saved. Or, to print it directly, just double-click the image file and press Ctrl+P (Windows) or Cmd+P (Mac).

🤔 Not Sure Which Method to Use?

Here's a quick guide to picking the right approach:

  • Using ChatGPT (Teacher Plan or Plus)? → Ask it to generate a downloadable Word or PDF file directly.
  • Using Google Gemini? → Ask for a Google Doc — it opens right in Google Drive and is ready to print. (Gemini doesn't create Word docs — Google Doc is your best option here.)
  • Using Claude? → Ask it to create a Word document or HTML file — it'll give you the file right in the chat. (Downloading Word or PDF files requires a Pro plan or higher.)
  • Using Microsoft Copilot? → Ask it to create a Word document or PDF — it's the most native Microsoft tool for this. If you have Copilot in Word, use it right there.
  • Need it to look exactly right on any device? → Ask for an HTML file and print from your browser.
  • Saving an AI-generated image? → Right-click → Save image as.
